A new update for "God of War Ragnarok" has been announced, version 5.05, which includes bug fixes and performance improvements. "God of War Ragnarok" will be released on PC on September 19, 2024, and will feature support for ultra-wide screens and performance-enhancing technologies like DLSS and FSR. Santa Monica Studios has announced a set of new abilities for Kratos, including special moves like "Frost Awaken" and "Whiplash". The PC version will also include additional content such as the "New Game+" mode and the "Valhalla" expansion, which offers a new roguelite experience. Players who pre-purchase the game will receive a set of cosmetic items for Kratos and Atreus, including the "Risen Snow" armor.
